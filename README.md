About the Project

The Niche Academy website was designed to provide a clean, engaging, and user-friendly experience while reflecting the school's commitment to academic excellence.

The website enables visitors to:
 Learn about the school's vision and mission
 Explore academic programs
 View school facilities
 Meet the teaching team
 Access admission information
 Contact the school easily
 Navigate seamlessly across all devices

Getting Started

Clone the repository:

```bash
git clone https://github.com/yusufabdhadeel-bot/The-Niche-Academy.git
```

Navigate to the project folder:

```bash
cd the-niche-academy
```

Open `index.html` in your preferred browser, or use the **Live Server** extension in Visual Studio Code for local development.
